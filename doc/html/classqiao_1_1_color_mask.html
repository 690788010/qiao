Detailed Description

ColorMask类包含是否开启颜色缓冲颜色分量的写操作的控制信息.

OpenGL的glColorMask函数用于开启或关闭颜色缓冲各个颜色分量的写操作， ColorMask类用于存放glColorMask函数的参数信息。
