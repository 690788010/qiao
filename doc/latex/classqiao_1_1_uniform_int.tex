\doxysubsection{Detailed Description}
Uniform\+Int类是\+Uniform\+Generic类的子类，它用于表示着色器中整数类型的\+Uniform变量。 Uniform\+Int类实现了\+ICleanable接口，从而为外部提供了值同步到\+GPU的clean函数。
