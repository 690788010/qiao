\doxysubsection{Detailed Description}
Color\+Mask类包含是否开启颜色缓冲颜色分量的写操作的控制信息. 

Open\+GL的gl\+Color\+Mask函数用于开启或关闭颜色缓冲各个颜色分量的写操作， Color\+Mask类用于存放gl\+Color\+Mask函数的参数信息。
